# Tools Notifikasi Suara Jam Berbicara
Ini adalah sebuah tools HTML sederhana (cukup simpan sebagai jam-notifikasi.html lalu buka di browser). Ia akan mengumumkan waktu dengan suara otomatis sesuai pengaturan: bahasa (Indonesia/Jawa/Sunda/English), format 24/12 jam, sertakan hari-tanggal-tahun opsional, dan interval (15/30/60/120/180/240 menit).

Tip: <br>
Klik “Mulai” sekali agar izin suara aktif. Tab harus tetap terbuka agar notifikasi berjalan.
